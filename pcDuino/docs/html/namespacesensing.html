def sensing.sendData (   sensorData)

Here we actually ship off the information to the server.

This function accepts a dictionary input of the data from the sensors, then inserts that dictionary into a different dictionary that wraps it with the board's identity and the time of the data collection.

Parameters
sensorDataA dictionary of the various sensor types and their values
Returns
Returns true on successful upload, and false otherwise
def sensing.setupSensors ( )

Any sensors that are available will be added to a list that gets polled later for their readings.
